# SEO Writer Skill

## Purpose
Write a complete, SEO-optimized article that ranks and converts.

## Writing Principles
- Lead with value — hook the reader in the first paragraph
- Use primary keyword in: first 100 words, title, at least 2 H2s, conclusion
- Natural keyword density: 1-1.5% for primary, 0.3-0.7% for secondary
- Short paragraphs (2-4 sentences max)
- Use bullet points and numbered lists for scanability
- Include real numbers, statistics, and examples
- Write for E-E-A-T signals

## Structure Requirements
- Introduction: Hook + promise + primary keyword
- Body: Follow outline structure strictly
- Conclusion: Summary + CTA
- FAQ: Detailed answers, minimum 50 words per answer

## Quality Checklist
- [ ] Primary keyword in title, intro, H2s, conclusion
- [ ] No keyword stuffing (natural flow)
- [ ] Include a comparison or data table wherever it helps scanability (pricing tiers, feature comparisons, pros/cons, step timelines, stat roundups)
- [ ] Tables must use full HTML: <table><thead><tr><th>...</th></tr></thead><tbody><tr><td>...</td></tr></tbody></table>
- [ ] Wrap every table in <div class="table-wrap"> for mobile scroll
- [ ] All H tags properly nested
- [ ] FAQ schema compatible format
- [ ] CTA is specific and action-oriented
