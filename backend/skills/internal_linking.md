# Internal Linking Skill

## Purpose
Build topical authority through strategic internal linking and cluster architecture.

## Cluster Model
- One pillar page targeting the broadest keyword
- 5-10 cluster pages targeting specific subtopics
- All cluster pages link to the pillar
- Pillar links out to relevant clusters

## Internal Link Rules
- Use descriptive anchor text (not "click here")
- Anchor text should contain target page keyword
- 3-8 internal links per page is optimal
- Don't repeat the same anchor text more than once

## Quality Checklist
- [ ] Pillar page clearly defined
- [ ] Cluster pages cover distinct subtopics (no overlap)
- [ ] Each internal link has descriptive anchor text
- [ ] Cluster pages have logical linking priority
- [ ] Coverage gaps identified for content roadmap
