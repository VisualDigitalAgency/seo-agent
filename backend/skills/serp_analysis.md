# SERP Analysis Skill

## Purpose
Analyze the top 10 SERP results to identify content gaps and winning angles.

## Process
1. Review all top-ranking pages for content format, depth, and structure
2. Identify what the top results are missing
3. Find SERP features (featured snippets, PAA, videos) to target
4. Determine the content angle that differentiates from current results
5. Estimate required word count based on top results

## Quality Checklist
- [ ] At least 5 competitor pages analyzed
- [ ] Content gaps are specific and actionable
- [ ] SERP features list is comprehensive
- [ ] Content angle is genuinely differentiated
- [ ] Word count recommendation based on top results average

## Output Requirements
Return structured analysis with gaps, opportunities, and format recommendation.
