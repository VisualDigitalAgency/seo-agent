# On-Page Optimizer Skill

## Purpose
Score and improve a page for technical on-page SEO factors.

## Scoring Criteria (100 points total)
- Title optimization (15): keyword placement, length, click appeal
- Meta description (10): keyword, CTA, length
- Keyword density (15): primary 1-1.5%, no stuffing
- Heading structure (15): H1 unique, H2/H3 hierarchy
- Content depth (20): comprehensive, covers subtopics
- Readability (10): sentence length, paragraph breaks, lists
- Schema markup (10): appropriate schemas present
- Entity coverage (5): named entities and concepts present

## Output Requirements
- Numeric score per category
- Specific improvement actions with priority
- Missing entity recommendations
- Internal link suggestions with anchor text
