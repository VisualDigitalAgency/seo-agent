# Keyword Research Skill

## Purpose
Identify the optimal keyword set for maximum organic traffic and ranking potential.

## Process
1. Analyze the primary topic for search volume and difficulty signals
2. Identify 3-7 secondary keywords with strong relevance and lower competition
3. Find 5-10 long-tail variants that capture specific intent
4. Identify LSI keywords for semantic depth
5. Classify search intent (informational/commercial/transactional/navigational)

## Quality Checklist
- [ ] Primary keyword has clear commercial or informational value
- [ ] Secondary keywords don't cannibalize primary
- [ ] Long-tail keywords represent real user queries
- [ ] Intent classification aligns with content type
- [ ] Competitor domains identified from SERP

## Output Requirements
- Structured JSON with all keyword categories
- Intent classification with explanation
- Difficulty and volume estimates
- Content type recommendation
